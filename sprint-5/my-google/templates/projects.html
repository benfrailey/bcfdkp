Concert Tracker

Concert Tracker On Github

My first big project that I completed was a concert tracker application. I love music, and so one of my first programming ideas was an application that can track all of the concerts that I've been to and let me save pictures and memories from each of the concerts. At the time, I had no idea how to actually make the application a reality, but once I took a class in Java at Mizzou, I learned everything that I needed to know to make this application a reality.



iCook

iCook On Github

It was right around Thanksgiving when I was finishing up my Concert Tracker application. I was with my extended family for Thanksgiving, and an uncle of mine who loves cooking asked me if it would be possible to make an application that would accept recipes, parse it to find the time it takes for each instruction, and then tell you what time each instruction needs to happen in order to have the meal prepared in time. I thought it was a really good idea, and so I began working on my new application, iCook (I always planned on coming up with a better name but never got around to it). I decided to make iCook as a Christmas present for my aunt and uncle. I finished the app just in time for my family's Christmas celebrations (The timeline was so close that I was working on it up until an hour before my extended family arrived). This was the first application I designed for someone else and I had a ton of fun making it!
